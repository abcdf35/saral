Ejemplo del Box Model de CSS

¡Contenido de la caja!

Componentes del Box Model:

Contenido:

Es el área donde se muestra el contenido real del elemento (texto, imágenes). Su tamaño se controla con `width` y `height`.

Padding:

Es el espacio transparente que rodea el contenido, pero aún está dentro del borde. Se define con `padding`.

Border:

Es la línea visible que rodea el padding y el contenido. Se define con `border`.

Margin:

Es el espacio transparente fuera del borde, que separa esta caja de otros elementos. Se define con `margin`.

box-sizing:

Propiedad crucial. Por defecto (`content-box`), `width` y `height` solo afectan al contenido, sumando padding y border al tamaño total. Con `border-box`, `width` y `height` incluyen padding y border, lo que simplifica mucho el diseño.
